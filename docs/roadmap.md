# Studiobuild Roadmap

## Phase 1: Static MVP

The current build covers the app shell and pipeline experience with local data only. This is the fastest route to a free hosted prototype on GitHub Pages.

- Dashboard shell
- Project model
- Inspiration capture
- Idea generation simulation
- Synopsis, treatment, short story, and script drafting simulation
- Basic script editor
- Breakdown and export gates
- Admin bypass behavior

## Phase 2: Backend Foundation

GitHub Pages cannot host private backend logic, so this phase adds a serverless API.

- Supabase project
- Auth provider setup
- PostgreSQL schema
- Serverless generation endpoints
- Admin email role assignment
- API-level feature gates
- Version history

## Phase 3: Paid Features

- Stripe checkout
- Stripe webhooks
- Billing portal
- Export tools
- Scene, character, location, prop, wardrobe, and beat breakdowns
- Shot list generation

## Phase 4: Premium Visuals

- Premium image endpoint
- Scene reference images
- Character reference images
- Storyboard panel generation
- Storyboard PDF export
- Drag-and-drop storyboard ordering
